What do the methods mean?

Each site in the list shows a method label that describes how it implements dark mode support. Here's what each one means:

  • MediaQuery — Uses the prefers-color-scheme CSS media query to automatically detect the user's dark mode preference. This is the recommended approach.
  • ColorScheme — Uses the color-scheme CSS property to declare support for dark mode.
  • JavaScript — Uses JavaScript to toggle between light and dark themes.
  • DarkOnly — The site only has a dark theme. There is no light mode.
  • OptIn — Users can manually enable dark mode through the site's UI.
  • Unknown — The method used to implement dark mode hasn't been determined.
